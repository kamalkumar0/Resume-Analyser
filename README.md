# 🧾 Resume Analyzer (AI + NLP Based)

This project is an AI-based Resume Analyzer that helps users understand how strong their resume is and whether it is ATS-friendly or not.

The idea behind building this project was simple — many students and job seekers don’t know why their resumes get rejected. So I created a tool that analyzes resumes using NLP techniques and gives feedback like extracted skills, score, and improvement suggestions.

🔗 Live Demo: https://huggingface.co/spaces/kamalsharma0/Resume_Analyser2.0](https://huggingface.co/spaces/kamalsharma0/Resume_Analyzer
🔗 LinkedIn: https://www.linkedin.com/kamalkumar0

---

## 🚀 What This Project Does

* Upload your resume (PDF)
* Extract important skills and keywords
* Analyze resume content using NLP
* Generate a resume score
* Provide suggestions to improve the resume
* Show how ATS systems might evaluate the resume

The goal is to help users improve their chances of getting shortlisted.

---

## ✨ Features

* Resume upload through web interface

* Resume scoring system
* ATS compatibility insights
* Improvement suggestions
* Simple and clean UI

---

## 🧠 How It Works (Basic Flow)

1. User uploads a resume
2. Text is extracted from the PDF/DOCX/JPG/PNG file
3. NLP libraries process the content
4. Skills and keywords are identified
5. Score and suggestions are generated
